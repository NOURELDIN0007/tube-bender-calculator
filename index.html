How the math works (Swagelok measure-bend method)
marking convention: Each MARK = the VERTEX of a bend (intersection of the two leg centerlines). Align the mark with the bender's reference, then bend to the target angle. cut length: L_total = Σ legs − Σ adjustments adjustment per bend (gain) from MS-13-43 p.23 table when (OD, R, θ) is tabulated; otherwise: 2·R·tan(θ/2) − R·θ mark positions on straight tube (from end A): M1 = leg_1 M2 = M1 + leg_2 − adj_1 M3 = M2 + leg_3 − adj_2 … L_total = M_last + leg_last − adj_last end-leg constraint: leg ≥ Lmin + R·tan(θ/2) S-offset (MS-13-43 p.11): L_diag = O × A, where A: 22½° → 2.613 30° → 2.000 45° → 1.414 60° → 1.154
